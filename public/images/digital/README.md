# 數位走讀互動館圖片放置說明

請依照互動主題資料夾放圖片，建議檔名如下：

- `cover.jpg`：入口卡片封面圖
- `gallery-01.jpg`、`gallery-02.jpg`：內容照片
- 走讀路線或地圖可命名為 `map.jpg`

互動主題資料夾：

- `chilan-game`：赤蘭溪闖關遊戲
- `chilan-walk`：赤蘭溪走讀
- `early-life`：赤蘭溪早年生活型態
- `puzi-medical-walk`：朴子醫療走讀

對應文字、連結、走讀站點資料請到 `outputs/site/content-data.js` 修改 `digitalTours` 區塊。
