# 社團記錄圖片放置說明

請依照社團資料夾放圖片，建議檔名如下：

- `cover.jpg`：社團卡片封面圖
- `gallery-01.jpg`、`gallery-02.jpg`：社團活動紀錄照片
- 學員作品可用 `work-01.jpg`、`work-02.jpg`

社團資料夾：

- `wood-repair`：嘉邑木工修繕社
- `saxophone`：薩克斯風社
- `dance`：多元舞蹈社
- `enka`：日本演歌說唱社
- `qigong`：太極十八式氣功暨身心調理社
- `sketch`：嘉憶速寫社

對應文字資料請到 `outputs/site/content-data.js` 修改 `clubs` 區塊。
