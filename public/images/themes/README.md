# 地方知識主題館圖片放置說明

請依照主題資料夾放圖片，建議檔名如下：

- `cover.jpg`：主題卡片封面圖
- `gallery-01.jpg`、`gallery-02.jpg`：主題內頁照片牆圖片
- 需要更多圖片時可接續 `gallery-03.jpg`

主題資料夾：

- `chilan-river`：赤蘭溪流域文化
- `puzi-medical`：朴子醫療地景
- `embroidery`：刺繡文化
- `coastal`：海區地方創生
- `food-agri`：食農教育與地方產業

對應文字資料請到 `outputs/site/content-data.js` 修改 `themes` 區塊。
